1.什么是面向对象？ 面向对象和面向过程的区别

​ 面向对象是基于面向过程的编程思想，由执行者变为指挥者，在现实生活中每一个物体都可以归位一类事物，而每一个个体都是一类事物的实例。面向对象的编程是以对象为中心，以消息为驱动

区别

​ 1.编程思路不同 ：面向过程以实现功能的函数开发为主，而面向对象首先要抽象出类，属性 ，以及方法，然后通过实例化，执行方法来完成功能。

          2. 封装性，都具有封装性看，但是面向过程是封装的是功能，面向对象封装的是数据和功能。
             3. 面向对象具有继承性和多态性 面向过程没有
